---
description: 智能提交助手：自动更新文档、生成提交信息并推送
---

1. **分析与文档更新**
   查看当前的代码变更 (`git diff` 或 `git status`)。
   - 如果完成了某个功能点，请自动更新 `FeatureTracker.md` 中的相关条目。
   - 如果修改了核心设计或架构，请检查并更新 `Design.md` 或 `contexts/context.md`。
   - *提示：先完成文档的修改保存，再进行下一步。*

2. **暂存更改**
   将所有修改（包括刚刚更新的文档）添加到暂存区。
   // turbo
   `git add .`

3. **生成并确认提交信息**
   根据变更内容，生成符合项目规范的中文提交信息。
   - 格式：`<类型>: <描述>`
   - 常用类型：`feat` (新功能), `fix` (修复), `refactor` (重构), `docs` (文档), `chore` (杂项)
   - **交互点**：必须向用户展示拟定的提交信息，并获得用户允许后才继续执行。

4. **提交代码**
   使用确认后的提交信息执行提交。
   `git commit -m "确认后的提交信息"`

5. **推送到远程仓库**
   // turbo
   `git push`
